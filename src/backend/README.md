# Backend Module

**Purpose**: Strictly houses the functional intelligence layer of PrepoAI. 

**Contents**:
- `ingestion.py`: PyPDF and PPTX text extraction utilities.
- `embeddings.py`: Cohere FAISS vectorization.
- `rag_engine.py`: LangChain History-Aware conversational RAG pipeline.
- `prompts.py`: The static master templates dictating behavior constraints.
